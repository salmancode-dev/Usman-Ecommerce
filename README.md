# 🛒 Usman E-Commerce Website

A modern, responsive, and user-friendly multi-page E-Commerce website built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. The project provides a complete online shopping experience with product categories, authentication pages, shopping cart, wishlist, checkout, and responsive layouts.


# ✨ Features

## 🛍️ Shopping Experience

- Responsive Home Page
- Product Categories
- Product Details
- Shopping Cart
- Wishlist
- Checkout Page
- Category Browsing

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/salmancode-dev/Usman-Ecommerce.git
```

Move into the project

```bash
cd Usman-Ecommerce
```

Open the project

Simply open:

```text
index.html
```

Or use VS Code Live Server.
